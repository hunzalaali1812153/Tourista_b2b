Deployment Steps:

Install React (npm install) on Web folder
Install Dependencies (npm install) on Backend folder

To run Web Front End - npm start
To run Backend - npm start

Install android emulator to run the app

To run Database:

Create mySQL database on XAMPP
Name the database 'tourista_b2b' and upload all the tables
